Desde el repositorio

Ejecuta el código fuente directamente con Python. Sin binarios firmados, sin advertencias del sistema.

① Clona o descarga el repositorio

git clone https://github.com/alejandroasc96/PiloPress.git

② Crea el entorno e instala dependencias

python -m venv .venv
.venv\Scripts\activate # Windows
pip install -r requirements.txt

③ Lanza la aplicación

python app.py

Procesamiento Flexible

Soporta escaneos recursivos de carpetas enteras, ejecuciones de una sola foto o la potente cola de selección múltiple con cálculo inteligente de rutas de salida.

⚡

Compresión Fina de Códecs

Ajusta de forma matemática el procesamiento. Configura el nivel de esfuerzo de la CPU (0 a 6), activa compresión sin pérdidas (*lossless*) nativa para WebP o fuerza escalados de píxeles máximos de forma proporcional.

🛡️

Privacidad y Limpieza EXIF

Elimina de forma automatizada los metadatos ocultos e innecesarios de tus imágenes (coordenadas GPS, modelo del dispositivo, fecha de captura). Menos peso, más seguridad.

🔄

Flujo de Trabajo Seguro

Trabaja con tranquilidad. El modo "En el sitio" optimiza el directorio original creando previamente un backup limpio en la carpeta interna `raw-imagenes` de respaldo.

Flujo en 3 pasos rápidos

1

Carga el Origen

Suelta una carpeta o arma una selección múltiple de imágenes en cola.

2

Modula el Códec

Elige WebP, JPEG o PNG y personaliza la calidad y las dimensiones del lado máximo.

3

Ejecuta en un Hilo

Pulsa Iniciar. La app procesará en paralelo sin bloquear ni congelar el software.
